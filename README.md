# Number Theory Graduation Project

## Overview

This repository contains my undergraduate graduation project on classical number theory, focusing on:

* Wilson's Theorem
* Fermat's Little Theorem
* Euler-Fermat Theorem

The project includes academic translation, mathematical typesetting and presentation materials prepared using LaTeX and Overleaf.

## Technologies

* LaTeX
* Overleaf
* Beamer
* amsmath
* tcolorbox
